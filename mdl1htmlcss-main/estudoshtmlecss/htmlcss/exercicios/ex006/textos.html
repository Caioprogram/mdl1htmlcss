Principais formatações de texto

Negrito/Destaque

Frase que possui negrito para destacar uma ou mais palavras usando a tag B

Frase que possui negrito para destacar uma palavra usando a tag STRONG

Itálico/Ênfase

Frase que possui termo em itálico para destacar uma palavra usando a tag I

Frase que possui termo em itálico para destacar uma palavra usando a tag EM (semântica)

Ambas as tags funcionam, caso tu queira colocar algo entre alguma tag de forma rápida, pode-se selecionar
a frase eu texto que irá no meio da tag, clicar control+shift+p e selecione o termo wrap with abreviation
e então colocar a tag desejada para aquela frase ou texto

Marcando o texto

Pode-se marcar um texto usando a tag MARK

para mudar a cor da marcação, se usa o parâmetro style dentro da tag mark, sendo esse um parâmetro de css

para que todos os marks fiquem de outra cor, deve-se usar a tag style antes da tag head, definindo todos os marks para a cor desejada

tamanho do texto

criação de um texto grande e criação de um texto pequeno nesse parágrafo

no texto anterior se aumentou a letra usando as tags big e small, porem essas tags vão se tornar obsoletas,
então para contornar esse problema, pode se usar o parametro font size e defidir o tamanho da fonte do paragrafo
exemplo:

o tamanho dessa fonte é maior do que o padrão(tamanho 5)

o tamanho dessa fonte é o tamanho padrão(tamanho 3)

o tamanho dessa fonte é menor do que o padrão(tamanho 1)

pode-se fazer todo o texto ser de outra cor ao adicionar o parâmetro font color antes de todo o texto

são 4 parâgrafos escritos dentro de um parametro font color, e todos os quatro com as letras verdes
